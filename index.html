Featured Projects

Windows Server 2012 R2 Domain Configuration (Operating Systems I)

I explored the functionality of Windows Server 2012 R2 by configuring a complete domain service environment. The project involved:

  • Setting up a domain service for a simulated company environment
  • Configuring DNS and connecting a Windows 7 Professional computer to the server's network
  • Creating and managing user accounts with specific group permissions (Purchasing and Sales) and implementing folder access controls
  • Developing and hosting a custom HTML webpage using Internet Information Services (IIS), made accessible via www.itsme.com
  • Utilizing key administrative tools such as Active Directory Domain Services (AD DS), Domain Name System (DNS), Dynamic Host Configuration Protocol (DHCP), and Internet Information Services (IIS)

This project demonstrated my ability to configure and manage complex server environments, showcasing skills in network administration, user management, and web hosting within a Windows Server ecosystem.

Event Organizer Application (Programming Languages)

We developed a comprehensive mobile application using Flutter and Firebase, designed to streamline event planning and management. Key features include:

  • Intuitive event creation and management interface
  • Seamless calendar integration for efficient event tracking
  • Robust guest list management functionality
  • Detailed user expense tracking for improved budget control
  • Customizable event templates and themes
  • Collaborative budgeting tools for group events
  • Integration with user's personal finances for comprehensive financial management

This project showcased our proficiency in mobile app development, database integration, and user-centered design. We created a practical tool that not only facilitates event organization but also provides comprehensive financial management features, making it a versatile solution for both personal and professional event planning.

Technical Service Management System (Database II)

We engineered an efficient system for a technical support company using SQL Server and C# with Visual Studio. The system comprehensively manages:

  • Client contracts and diverse service packages
  • Dynamic technician assignments based on client needs and real-time technician availability
  • Detailed inventory tracking of tools and materials used in service provision
  • Complex financial transactions including monthly billing, package subscriptions, and itemized service charges
  • Company bank accounts, meticulously tracking income from invoices and various expenses such as purchases and payroll

Our project showcased advanced database design skills, including the creation of a robust database structure and implementation of stored procedures for CRUD operations across all tables, incorporating sophisticated auto-numbering calculations where necessary.

Hotel Information Systems Analysis and Recommendation (Information Systems)

We conducted a comprehensive analysis and implementation plan for Hotel Nacional S.A., focusing on enhancing their existing ERP system (Skill4IT) with complementary information systems. Our project encompassed:

  • Thorough evaluation of the hotel's current information systems, identifying strengths and areas for improvement
  • In-depth analysis of various information system types, including ERP, CRM, SCM, KWS, LMS, ES, Machine Learning, BI, and DSS to determine the most suitable solutions for the hotel's specific needs
  • Recommendation of Salesforce Service Cloud as a complementary CRM system to work alongside the existing Skill4IT ERP
  • Development of a detailed implementation plan for integrating the recommended systems with Skill4IT

This project demonstrated our ability to analyze complex business needs, recommend appropriate technological solutions, and create a comprehensive plan for successful implementation and integration of multiple information systems in a real-world business environment.
