Hotel Information Systems Analysis and Recommendation (Information Systems)

  • Recommendation of Salesforce Service Cloud as a complementary CRM system to work alongside the existing Skill4IT ERP
  • Development of a detailed implementation plan for integrating the recommended systems with Skill4IT, including:
    • Phased deployment strategy to minimize operational disruptions
    • Comprehensive employee training programs
    • Integration architecture design using middleware and APIs for seamless data flow between systems
    • Custom configuration of the new systems, including creation of personalized objects and fields
    • Implementation of interactive features like chatbots and mobile access to enhance customer service
    • Establishment of automated workflows to improve operational efficiency
  • Post-implementation evaluation and continuous improvement strategies

This project demonstrated our ability to analyze complex business needs, recommend appropriate technological solutions, and create a comprehensive plan for successful implementation and integration of multiple information systems in a real-world business environment.
